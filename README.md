# short-url_go
### Сервис по сокращению ссылок (в разработке)
Работает по http

### Содержит в себе
- Запуск с флагами
- Тестирование handlers
- Логирование

### Флаги
- `-l` - уровень логирования например (`info`)
- `-a` - адрес и порт запуска например (`localhost:8080`)
- `-b` - базовый URL ответа например (`http://localhost:8080`)
